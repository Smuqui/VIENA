Connect to the GPS
 Adquires velocity and position information from the gps system
 usage : gps(comPort<port>",
 dataFile="<your datafile>",
 logFile="<your logfile>",
 baudRate=9600,
 loglevel=<logging level int>)
 Default values:
     comPort="/dev/ttyUSB0"
     dataFile="./data/out.csv"
     logFile="output.log"
     baudRate=9600
     logFevel=20

     loglevels are defined as:
     -----------------------
     Level    | Numeric value
     -----------------------
     CRITICAL |  50
     ERROR    |  40
     WARNING  |  30
     INFO     |  20
     DEBUG    |  10
     NOTSET   |  0
     -----------------------
     check documentation of module logging for more info

 HW info:
 Receptor - Novatel Flexpak G2L-3151W
 Antenna - Novatel Pinwheel
